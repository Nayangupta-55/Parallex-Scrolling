# Interactive Parallax Scrolling Website

## Project Overview

This project is an interactive parallax scrolling web experience designed to create a visually immersive journey for users. It transforms traditional webpage navigation into an engaging storytelling experience using layered animations and dynamic transitions.

The website consists of multiple full-screen sections where background and foreground elements move at different speeds during scrolling, creating a realistic depth effect.

---

## Objective

The objective of this project is to demonstrate how modern web technologies can be used to build interactive, responsive, and visually appealing web experiences using parallax scrolling techniques.
